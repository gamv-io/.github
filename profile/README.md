## 🚀 Build Games for Gamvio

```bash
npx create-gamvio-game my-game
cd my-game && npm install && npm run dev
```

Get your SDK keys at [dev.gamv.io](https://dev.gamv.io) and start building in minutes.

## 🛠 Tech Stack

```
Frontend    Next.js 15 · React 19 · Tailwind CSS
Backend     Go · Fiber · PostgreSQL · Redis
Blockchain  Xphere (EVM) · Thirdweb SDK
Auth        Google · Kakao OAuth
SDK         TypeScript · HMAC Anti-Cheat
```

## 🎯 Features

- **75+ Mini-Games** — Arcade, puzzle, board, card, strategy
- **Anti-Cheat Scoring** — HMAC-SHA256 signed score submission
- **Token Economy** — POINT rewards for gameplay and quests
- **Daily Quests** — Check-in streaks and achievement system
- **Leaderboards** — Global and weekly rankings per game
- **Developer SDK** — Build and publish your own games
- **Test Mode** — Local development with instant test tokens
